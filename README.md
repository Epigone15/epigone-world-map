# epigone-world-map

Une carte interactive du monde où tu peux cliquer sur les pays pour les colorier.

## 🌍 Utilisation

1. Ouvre [https://epigone15.github.io/epigone-world-map/](https://epigone15.github.io/epigone-world-map/).
2. Choisis une couleur dans la palette.
3. Clique sur un pays pour le colorier.
4. Utilise le bouton "Réinitialiser" pour effacer toutes les couleurs.

## 🛠️ Personnalisation

- Pour ajouter plus de pays, modifie le tableau `countries` dans le fichier `index.html`.
- Pour changer les couleurs, modifie le tableau `colors` dans le fichier `index.html`.
